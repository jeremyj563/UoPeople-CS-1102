So far the course content is instructing us to setup the Eclipse interactive development environment (IDE) to use for all of our code editing. If you want to try a different editor that has more modern features and a much higher adoption rate, then I would strongly recommend installing Microsoft Visual Studio Code (VSCode). Below are some simple instructions for getting setup depending on your computer's operating system (I apologize in advance, I do not own a Mac so I am unable to provide exact steps for macOS).

Microsoft Windows

    Install the Chocolatey package manager
    • open powershell and run:
    • PS > Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
    Install OpenJDK 11 and VSCode
    • PS > choco install -y ojdkbuild11 vscode

Ubuntu Linux

    Install OpenJDK 11 using APT
    • $ sudo apt install -y openjdk-11-jdk
    Install VSCode using Snap
    • $ sudo snap install --classic code

Important:

After installation is complete simply open VSCode and navigate to:

  • Extensions > search 'java' > Java Extension Pack > Install
This will the Java extensions pack for VSCode that will add support for all of the necessary features for working with Java such as debugging, code navigation, auto completion, refactoring, code snippets, etc...
