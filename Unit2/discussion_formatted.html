Example 1: while

In this example we can see the difference between while and do-while. With while the boolean expression is evaluated before iterating the block but with do-while the boolean expression is not evaluated until after the first iteration of the block. We can also see one of the benefits of for is that we can define the iterating variable within the for statement itself.

// Example 1: while (will loop only once)
boolean loop = true; int count = 1;
while (loop) {
    System.out.println("While - count: " + count++);
    loop = false;
}

// equivalent do-while (will loop twice)
boolean loop = true; int count = 1;
do {
    System.out.println("Do-While - count: " + count++);
    loop = false;
} while (loop);

// equivalent for (with in-line variable delcaration)
boolean loop = true;
for (int count = 1; loop; count++) {
    System.out.println("For - count: " + count++);
    loop = false;
}

Example 3: for

Finally we have for. In this example we can see plainly how using for helps to not pollute the outer scope by, once again, declaring the iterating variable in-line.

// Example 3: for
for (int count = 10; count >= 5; count--) {
    System.out.println("For - count: " + count);
}
